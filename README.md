# <img src="https://color-tool-np.vercel.app/favicon.ico" width="25" /> ColorTool

[🔗 Preview](https://color-tool-np.vercel.app/)

ColorTool is a Nuxt 4 app for quick color workflows:
- Color format conversion
- Gradient generation
- Avatar generation
- WCAG contrast checking
- Extract color palette from image

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
